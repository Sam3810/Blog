Imagenes

Como ya se sabe las imagenes pueden llegar a ser esenciales en una web, hace que se vean mas lindas tus paginas, da a entender al usuario de lo que se esta hablando entre muchos otros beneficios que nos pueden llegar a entregar, pero tiene su parte mala y es que muchas personas no saben que las imagenes pueden llegar a ser pesadas y al implementarlas en tu web pueden llegar a afectar el rendimiento significativamente, lo cual puede arruinar la experiencia del usuario, asegurate de usar herramientas como las que te muestro en este articulo.

Codigo muerto

El codigo muerto es una parte de codigo que esta siendo ejecutada pero sus resultados nunca se utilizan, seria como consumir una api sin usarla, esto puede llegar a afectar negativamente el rendimiento de nuestra web debido a que se esta ejecutando algo que no se utiliza, este codigo muerto se puede encontrar como en HTML, CSS y JavaScript.

Alojamiento web

Otra de las razones por las que puede que tu web tenga mal rendimiento se debe a que tengas un alojamiento de mala calidad, por lo cual es recomendable que te asegures de donde alojar tu web e investigar sobre este alojamiento.
