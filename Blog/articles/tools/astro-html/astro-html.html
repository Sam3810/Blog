Guía de migración de HTML a Astro

Autor: Genaro Bonavita

28 Junio 2023

Paso 1: Configuración inicial

Asegúrate de tener Node.js instalado en tu computadora. Puedes descargarlo desde el sitio web oficial de Node.js e instalarlo siguiendo las instrucciones proporcionadas.

Paso 2: Crear un nuevo proyecto Astro

Abre tu línea de comandos o terminal y crea una nueva carpeta para tu proyecto Astro.

Navega hasta la carpeta del proyecto en la línea de comandos.

Ejecuta el siguiente comando para crear un nuevo proyecto Astro:

npx create-astro nombre-del-proyecto

Esto creará una nueva carpeta con el nombre del proyecto y configurará una estructura básica para tu proyecto Astro.

Paso 3: Migrar archivos HTML

Copia todos los archivos HTML de tu proyecto anterior y pégalo en la carpeta src de tu nuevo proyecto Astro.

Si tienes archivos HTML en subcarpetas, mantén la misma estructura de carpetas en la carpeta src de tu proyecto Astro.

Paso 4: Convertir archivos HTML a componentes Astro

Abre cada archivo HTML en la carpeta src y renómbralo con la extensión .astro.

Astro utiliza una sintaxis similar a JSX, por lo que puedes utilizar JavaScript dentro de tus componentes Astro para lógica adicional si es necesario.

Paso 5: Revisar y ajustar

Una vez que hayas convertido todos los archivos HTML a componentes Astro, revisa tu proyecto para asegurarte de que todo se vea y funcione correctamente.

Ajusta cualquier referencia a rutas de archivos, como enlaces a hojas de estilo CSS o imágenes, para que apunten a las ubicaciones correctas en tu proyecto Astro.

Paso 6: Compilar y ejecutar tu proyecto Astro

Ejecuta el siguiente comando en tu línea de comandos para compilar y ejecutar tu proyecto Astro: 

npm run dev

Esto compilará tu proyecto y lo ejecutará en un servidor de desarrollo local.

Abre tu navegador web y ve a http://localhost:3000 para ver tu proyecto Astro en acción.
