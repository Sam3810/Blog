Menu hamburguesa

Autor: Samuel Vanegas Bueno

6 Julio 2023

Probablemente cuando creas tu web siempre le añades un header para que el usuario pueda navegar por la web sin mayor esfuerzo, normalmente este header esta posicionado horizontalmente y cuando se revisa en mobile, ¡se rompe! para solucionar esto hoy te enseñare a crear un menu hamburguesa.

HTML

Comencemos por el HTML, lo primero que debemos hacer es añadir 2 botones, uno que servira para abrirlo y el otro para cerrarlo, como probablemente tu header contenga una etiqueta nav debes añadir el boton de cerrar el menu dentro de este nav y el boton de abrir solamente lo dejas dentro del header.

CSS

Para el CSS vamos a llamar a los 2 botones anteriores de abrir y cerrar y les daremos un display none, ahora debemos crear una media querie a la cual le daremos un max-width del valor que deseemos y dentro de esta le daremos a los botones un display block, seguido de esto cambiaremos el estilo de tu nav como gustes, puedes tomar como referencia el siguiente codigo:

.nav { 
 display: flex; 
 align-items: end; 
 flex-direction: column; 
 padding-top: 20px; 
 padding-right: 10px; 
 gap: 10px; 
 border-radius: 10px; 
 position: absolute; 
 top: 0; 
 right: 0; 
 bottom: 0; 
 height: 100vh; 
 background-color: black; 
 box-shadow: 0 0 0 100vmax rgba(0, 0, 0, 0.5); 
 visibility: hidden; 
 opacity: 0; 
}

Asegurate de que tengas visibility: hidden; y opacity: 0; en el CSS del nav en la media querie, ahora debes crear una nueva clase, con el nombre que gustes y le vas a escribir el siguiente codigo:

.class { 
 visibility: visible; 
 opacity: 1; 
}

Esta clase la utilizaremos mas adelante.

JavaScript

Por ultimo, coronamos con JavaScript, lo que debes hacer es escribir el siguiente codigo:

let nav = document.querySelector('class o id') 
let abrir = document.querySelector('class o id') 
let cerrar = document.querySelector('class o id') 
 
abrir.addEventListener('click', () => { 
 nav.classList.add('classCSS') 
}) 
 
cerrar.addEventListener('click', () => { 
 nav.classList.remove('classCSS') 
})

En la parte donde dice class o id escribes el que la hayas añadido a los botones, y en donde dice classCSS es la clase que te dije que crearas en el CSS hace un momento, con esto hecho tu menu hamburguesa deberia funcionar, si hay algun problema por favor notificalo en el repositorio de GitHub.
